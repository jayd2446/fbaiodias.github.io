# my website

Based on [Harp](http://harpjs.com/), and the [Remedy](https://github.com/kennethormandy/hb-remedy) theme.

## Get started

```sh
yarn install

git checkout source

yarn start
```

Now visit [localhost:9000](http://localhost:9000) in your browser.

## Compile and deploy

```sh
yarn deploy # or ./deploy.sh
```
